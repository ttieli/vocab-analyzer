# **📘 英文书词汇等级分析服务流程简介**

## **🎯 服务目标**

将一本英文书（或任意英文文本）自动转化为：

* **各单词 / 词组的 ****CEFR 等级（A1–C2）**
* **对应 ****剑桥考试阶段（KET → PET → FCE → CAE → CPE）**
* **对应 ****雅思分数段（IELTS 3.0–9.0）**
* 并生成可视化统计与导出文件（CSV / JSON / 图表）

---

## **🪜 一、整体处理流程**

```
graph TD
A[上传英文书籍] --> B[文本清洗与分词]
B --> C[词形还原 + 词组识别]
C --> D[词汇匹配到剑桥词表 (A1–C2)]
D --> E[等级映射 → 雅思分数区间]
E --> F[统计分析与可视化输出]
F --> G[导出结果(JSON/CSV/HTML)]
```

---

## **🧩 二、主要处理环节说明**

### **1️⃣ 文本输入**

* **接收 **.txt**、**.pdf** 或 **.epub** 文件；**
* 提取纯英文文本；
* 去除标点、章节标题、非英文字符。

### **2️⃣ 分词与词形还原**

* 使用 **spaCy** / **nltk** 进行英文分词；
* 将单词统一为原形（如  *went → go* ）；
* 过滤停用词（a, the, of 等）。

### **3️⃣ 词组识别（可选）**

* 识别常见动词短语（look up, take off 等）；
* 识别高频搭配（make a decision, take part in 等）。

### **4️⃣ 等级匹配与映射**

* 匹配剑桥官方词汇表：
  * A2 Key (KET)
  * B1 Preliminary (PET)
  * B2 First (FCE)
  * C1 Advanced (CAE)
  * C2 Proficiency (CPE)
* 每个词汇被标注：
  * {"word": "develop", "level": "B1", "exam": "PET", "ielts": "5.0-5.5"}

### **5️⃣ 统计与分析**

* 统计各等级词汇占比；
* 生成难度曲线图（章节难度变化）；
* 标出超纲词（未出现在剑桥词表中的）。

### **6️⃣ 输出与展示**

* 导出结构化文件：
  * results.json**（含所有词汇详情）**
  * summary.csv**（等级分布统计）**
* 前端页面可视化展示：
  * 饼图：词汇等级分布
  * 折线图：章节难度趋势
  * 表格：高频词汇 + 等级标注

## **⚙️ 三、系统架构概览**

| **模块**             | **主要职责**   | **技术栈**       |
| -------------------------- | -------------------- | ---------------------- |
| **Frontend**         | 上传书籍、展示结果   | Vue / React + Plotly   |
| **API Backend**      | 接收文件并触发分析   | FastAPI / Flask        |
| **NLP Engine**       | 分词、还原、词汇匹配 | spaCy + Pandas         |
| **Level Dictionary** | 剑桥+雅思词汇映射    | CSV / JSON 词典库      |
| **Storage**          | 保存结果             | Local / SQLite / MinIO |
